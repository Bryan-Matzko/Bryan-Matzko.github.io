Reachable Point in Space Calculator

During my second year in University, I took a numerical methods course, one of our projects was to solve a problem that may be found in the real world. My group decided to take a 3 joint robotic arm and determine the angles the are would need to make in order for the tip of the arm to be at the specified location. Several simplifications/assumptions were made and MATLAB scripts were written to first determine the divergence regions for the matrix representation of the system. From my findings, I wrote a C# application was written that was able to take any point and determine all possible angles the joint could make to reach the desired position.

I took all the MATLAB code and re-wrote it in C#. I made my own functions to manipulate matrices, and solve systems of equations then wrote a console application and a web application to display my results in different forms for presentations. The original MATLAB code took over 24 hours to run a basic solution. My C# code was able to perform all of the same calculations in under 10 seconds. This then allowed me to increase the standards for my solutions and run more complicated scenarios. At one point I had a program that would take into account the error approximation the user wanted and the application would determine all angles withing the provided tolerance that provided a solution. This was outside the scope of our proposed project description and I removed the code for submission for the mandatory presentation my group did.
